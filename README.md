# Back-Concenso

Backend para sistema de gestión de gimnasio desarrollado con Node.js, Express y TypeORM.

## Tecnologías Principales

- Node.js
- Express
- TypeORM
- PostgreSQL
- TypeScript
- Docker

## Requisitos Previos

- Node.js 18 o superior
- Docker y Docker Compose
- PostgreSQL
